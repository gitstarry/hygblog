vue-cli

1.安装node.js

  1. mis安装：
    进入官网 选择LTS版直接安装下载，安装过程比较简单，一路next即可。

  2. zip安装
    以压缩包的格式安装，需要手动添加环境变量

  3. nvm安装
    如果安装了nvm，可以使用nvm install v（版本号）命令安装。

  4. 安装完成后：node -v若出现版本号代表安装成功

  5. node.js后面的版本都集成了npm（一个包管理工具），后面需要用到它来下载和管理依赖。

  6. 安装cnpm：npm install -g cnpm -registry=https://registry.npm.taobao.org
    因为很多资源所在服务器都在国外，为了提高下载速度和被墙的问题，推荐使用cnpm
    cnpm -v出现版本号代表安装成功了

  7. 可以选择安装nvm （node.js 的版本管理工具）

2.安装vue-cli

  1. 若之前没有安装过vue-cli 可以直接安装，若之前安装过低版本的vue-cli输入：npm uninstall vue-cli -g
  2. 接着输入：npm install -g @vue/cli 不加版本号安装的是最新版本
  3. 我们这次安装的是指定版本：cnpm install -g @vue/cli@3.11.0
  4. vue -V出现版本号代表安装成功

3.使用vue-cli搭建vue项目

  1. vue create+项目名称创建项目

  2. 搭建过程中的具体配置环节太多了这里跳过。如果讲了记不住的，移步谷歌，上面有很多相关博客。

  3. 给出成功截图

    到这里基本就成功了，我们可以执行它提示的命令试一下。

    看到这个vue大logo就代表你的vue项目运行起来了

4.安装Vetur扩展

可以输入@installed 查看是否安装了该扩展
